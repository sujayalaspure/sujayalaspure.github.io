## üõ† Installation & Set Up

1. Clone the repo

   ```sh
   git clone https://github.com/alaspuresujay/alaspuresujay.github.io.git
   ```

2. Install dependencies

   ```sh
   yarn
   ```

3. Start the development server

   ```sh
   yarn start
   ```
